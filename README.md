## 미니 팀 프로젝트 Crypto(가제)


### 프로젝트 주제

- 암호화폐 거래소인 업비트 API를 사용하여 암호화폐를 자동으로 매수/매도 하는 웹사이트

##### 변수명 설정
표기법은 각 언어별 관행을 따른다

- html/css
  - 스네이크 표기법  
    ex) main_view
  - 해당섹션의 대표 영역만 id로, 나머지는 class로 작성
  
- javascript
  - 카멜 표기법
    ex) mainView
변수명 작성은 분류가 큰 순서로 작성
ex) mainViewSelectBtn
